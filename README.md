##Project startup

####Frontend
-Pull from the main branch
-cd into the eco-ryde folder
-run the `npm install` command (must have node.js installed)
-run `npm start`

####Backend
-Pull from the working-branch branch
-cd into the backend folder
-run the `npm install` command (must have node.js installed)
-run `node server`
